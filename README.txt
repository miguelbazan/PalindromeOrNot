Problem Description

A palindrome is a phrase whose meaning is the same either in forward or reverse direction.
Write a program that reads a string containing a phrase and that displays a message that indicates if it is a palindrome or not.

NOTE: that the prhase may contain several words and can contain upper case and lower case letters.

Input

Read a phrase in each row. The program finishes when the phrase is the word END.

Output

For each row comes a YES if the phrase is palindrome or NO if the phrase isn't.
Sample Input

Somos o no somos
ser o no ser
Anita Lava La Tina
END

Sample Output

YES
NO
YES
